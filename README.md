# Bubble Sorting Algorithm

This is a visual demonstration of how a bubble sorting algorithm works, using JavaScript and jQuery. I created this project while learning how to code at [The Last Mile](https://thelastmile.org/).
  
## Live Demo

See it in action at <https://brett-buskirk.github.io/sort-bubble/>.

You can also simply open the `index.html` file in a browser window and watch the animation. Reload the browser window to watch the animation again with a different set of random values.
